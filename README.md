# cTrader

# Quick Start
1. pip3 install -r requirements.txt
2. Create .env file
3. Put the following
```
APP_CLIENT_ID="INPUT YOUR VALUE HERE"
APP_CLIENT_SECRET="INPUT YOUR VALUE HERE"
ACCESS_TOKEN="INPUT YOUR VALUE HERE"
```
4. To get the value, go to https://openapi.ctrader.com/apps
5. Click "Credentails"
6. Click "Sandbox"
7. You know how to get the values
8. After that, to start the script, run `python3 main.py`
